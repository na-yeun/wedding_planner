# 💍 Wedding Planner

## 📌 프로젝트 개요
본 프로젝트는 결혼을 앞둔 예비 신혼부부를 위한 웨딩 준비 협업 기반 관리 시스템을 개발하는 것을 목적으로 한다.  
결혼 준비 과정에서 발생하는 다양한 일정, 예산, 지출 정보를 체계적으로 관리하고, 사용자 간 원활한 커뮤니케이션과 정보 공유를 지원한다.  
이를 통해 준비 진행 상황을 실시간으로 확인하고 관리할 수 있도록 하여, 효율적인 의사결정과 협업을 가능하게 하는 것을 목표로 한다. 

---

## 🧭 분야 / 목적 / 대상

### ✔️ 분야
- 웨딩 준비 협업 기반 관리 시스템

### ✔️ 목적
- 예비 신혼부부가 예산, 지출, 일정, 체크리스트 등을 체계적으로 관리
- 사용자 간 원활한 정보 공유 및 협업 지원
- 결혼 준비 과정의 복잡한 의사결정 및 일정 관리 부담 감소
- 실시간 데이터 공유를 통한 커뮤니케이션 오류 최소화
- 준비 진행 상황을 한눈에 파악하여 효율적인 계획 수립 지원 

### ✔️ 대상
- 결혼 준비 중인 예비 신랑, 신부

---

## 👥 서비스 사용자

### 일반 사용자 기능
- 마이페이지 (개인 정보 관리, 짝꿍 관리)
- 결혼 예산 관리
- 결혼 지출 관리
- 일정 관리 (캘린더)
- 시기별 체크리스트 제공 및 수정

---

## 🚀 주요 기능

### 1. 회원가입
- 아이디, 비밀번호, 이름, 핸드폰번호, 결혼 예정일 입력
- 카카오 OAuth 회원가입 지원

### 2. 로그인
- 일반 로그인 및 OAuth 로그인 지원
- OAuth 토큰 만료 시 재인증
- 비밀번호 3회 오류 시 문자 인증
- 비밀번호 찾기 기능 제공

### 3. 마이페이지
- 짝꿍 정보 조회
- 짝꿍 신청 및 삭제
- 개인정보 수정 (비밀번호, 연락처, 결혼 예정일)

### 4. 체크리스트
- 체크리스트 작성 / 삭제
- 완료 체크 및 해제
- 메모 작성, 수정, 삭제 기능

### 5. 일정 관리
- 일정 등록 (명칭, 날짜, 시간 필수 / 장소, 메모, 알림 선택)
- 일정 수정 및 삭제
- 알림 기능 제공

### 6. 지출 관리
- 기본 카테고리 제공 (홀, 스드메, 혼주 등)
- 카테고리 추가 / 수정 / 삭제
- 결제 상태 관리 (계약금, 잔금 등)
- 결제 수단 관리 (현금, 카드 등)
- 예산 입력 및 실제 지출 비교 기능

---

## 🛠️ 사용 기술

- FullCalendar API 기반 일정 관리
- OAuth 기반 인증 시스템
- SweetAlert2 기반 UI 개선
